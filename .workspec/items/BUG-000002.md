# Summary

Rapidly clicking the checkout button can fire two subscribe requests.

# Steps to Reproduce

1. Open checkout.
2. Double-click "Subscribe" quickly.

# Expected Behavior

Only one request is sent; the button is disabled while pending.

# Actual Behavior

Two requests are sent, occasionally creating duplicate subscriptions.
