# Summary

Timeboxed comparison of candidate billing providers.

# Questions to Answer

- Which providers support our reconciliation model?
- What are the webhook reliability guarantees?

# Findings

Two providers meet the requirements; one has stronger idempotency guarantees on
webhooks, which fits the worker-based architecture better.

# Recommendation

Proceed with the provider offering idempotent webhook delivery. Implementation
is tracked under EPIC-000002.
